Coot 1.2 Released 1st April 2026


## Graphics

 o FEATURE: Side-by-side stereo mode (cross-eye and wall-eye) with
            configurable stereo angle

 o FEATURE: High-resolution screenshots now work for both basic and
            fancy rendering modes

 o FEATURE: Material properties added to instanced-object shaders

 o FEATURE: Anisotropic thermal ellipsoid "empty" representation
            with multi-ring display

 o FEATURE: Gaussian surface opacity control

 o FEATURE: Bond smoothness toggle

 o FEATURE: Shadows for "other meshes" (generic display objects)

 o FEATURE: Single-model view for CA+Ligands representation with
            colour-by-dictionary-and-chain

 o FEATURE: Distance labels display can now be toggled with
            set_show_distance_labels()

 o FEATURE: scale_up_graphics() and scale_down_graphics() now
            added to the API

 o CHANGE:  Improved default secondary structure colours

 o CHANGE:  Rainbow ribbons are less bright

 o CHANGE:  Aniso probability scale now goes to 0.99

 o CHANGE:  Ribbon coils restored with radius multiplier for spline
            representation

 o CHANGE:  End-caps added for worm representations

 o CHANGE:  Max SSAO kernel samples doubled to 512

 o CHANGE:  Reduced step size for front clipping plane changes

 o CHANGE:  Don't include hydrogen atoms in Local B-factor display

 o CHANGE:  x-ray Sharpen/Blur Map slider now displays the value

 o CHANGE:  Difference map peaks now shows closest atom not position

 o BUG-FIX: Now the Recover Session works as expected [Kevin Jude]

 o BUG-FIX: Fix the rotation jump after panning

 o BUG-FIX: Fix peptide straggle-bond when displaying alt-confs

 o BUG-FIX: Fix texture indexing in TextureMesh

 o BUG-FIX: Fix framebuffer rendering with transparency

 o BUG-FIX: Fix the map colour button in single-map properties dialog

 o BUG-FIX: Fix ribbon surface normal computation and spline frame
            orthonormality [Martin Noble]
